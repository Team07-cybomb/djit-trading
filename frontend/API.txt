
Authentication api---(overall)
---------------------
User Authentication:
---------------------
POST http://localhost:5000/api/auth/register - Register a new user

POST http://localhost:5000/api/auth/login - User login

GET http://localhost:5000/api/auth/me - Get current user profile (protected)

Admin Authentication:
-----------------------
POST http://localhost:5000/api/admin/auth/login - Admin login

GET http://localhost:5000/api/admin/auth/verify - Verify admin token

POST http://localhost:5000/api/admin/auth/logout - Admin logout (client-side token removal)

POST http://localhost:5000/api/admin/auth/setup - Create initial admin (setup only)

User Management APIs
---------------------
User Operations:
----------------
GET http://localhost:5000/api/users/me - Get current user profile

PUT http://localhost:5000/api/users/profile - Update user profile

POST http://localhost:5000/api/users/upload-profile-picture - Upload profile picture

GET http://localhost:5000/api/users/:id - Get user by ID

GET http://localhost:5000/api/users - Get all users (admin only)

Admin User Management
-----------------------
GET http://localhost:5000/api/admin/users - Get all users with search & pagination

GET http://localhost:5000/api/admin/users/:id/details - Get detailed user information

PUT http://localhost:5000/api/admin/users/:id/role - Update user role

DELETE http://localhost:5000/api/admin/users/:id - Delete user

PUT http://localhost:5000/api/admin/users/:id/profile - Update user profile (admin override)

GET http://localhost:5000/api/admin/users-stats - Get user statistics

Course Management APIs
------------------------

-------------------------
GET http://localhost:5000/api/courses - Get all courses

GET http://localhost:5000/api/courses/:id - Get single course details

Admin Course Operations
------------------------
GET http://localhost:5000/api/admin/courses - Get all courses (admin version)

POST http://localhost:5000/api/admin/courses - Create new course

GET http://localhost:5000/api/admin/courses/:id - Get single course (admin version)

DELETE http://localhost:5000/api/admin/courses/:id - Delete course

GET http://localhost:5000/api/course-content/:courseId - Get course contents (protected)

POST http://localhost:5000/api/course-content/upload - Upload course content (admin only)

PUT http://localhost:5000/api/course-content/:id - Update course content

GET http://localhost:5000/api/courses/:id/content - Get course content for enrolled users
